Tweaking content editing experience

EditorModelEventManager Events

The EditorModelEventManager class is used to emit events that enable you to manipulate the model used by the backoffice before it is loaded into an editor

EditorModelEventManager Events

Introduced in Umbraco 7.4 (January 2016)

Documented in November 2017

EditorModelEventManager Events

You can subscribe to following events

  • SendingContentModel
  • SendingMediaModel
  • SendingMemberModel
  • SendingUserModel
